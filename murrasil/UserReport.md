# 📰 مُراسِل — دليل الاستخدام

> منصة متابعة أخبار التكنولوجيا والذكاء الاصطناعي بالعربية

***

## ⚡ تشغيل سريع (أول مرة)

### الخطوة 1 — احصل على مفتاح Gemini AI مجاناً

1. افتح الرابط: <https://aistudio.google.com/app/apikey>
2. سجّل دخول بحساب Google
3. اضغط **Create API Key**
4. انسخ المفتاح (ستحتاجه في الخطوة التالية)

***

### الخطوة 2 — الإعداد (مرة واحدة فقط)

```
انقر مزدوجاً على: setup.bat
```

- سيثبّت المكتبات تلقائياً
- سيفتح ملف `.env` لك — الصق مفتاح Gemini هكذا:

```
GEMINI_API_KEY=AIzaSy...مفتاحك_هنا
```

- احفظ الملف وأغلقه

***

### الخطوة 3 — التشغيل اليومي

```
انقر مزدوجاً على: start.bat
```

✅ سيفتح المتصفح تلقائياً على: <http://127.0.0.1:8000>

***

### إيقاف التطبيق

```
انقر مزدوجاً على: stop.bat
```

أو اضغط **Ctrl+C** في نافذة الخادم المفتوحة

***

## 🗂️ شرح الواجهة

### التبويبات الثلاث

| التبويب | الوصف |
|---|---|
| 🆕 أخبار جديدة | أخبار لم تراجعها بعد |
| ✅ تمت الموافقة | أخبار كتبت عنها مقالاً |
| ❌ مرفوضة | أخبار قررت تجاهلها |

### أزرار كل خبر

- **✅ كتابة الخبر** — يولّد مقالاً احترافياً بالعربية تلقائياً
- **❌ رفض** — يخفي الخبر وينقله لقائمة المرفوضات

***

## ⚙️ الإعدادات

افتح قائمة الإعدادات من أيقونة ⚙️ في الشريط العلوي:

- **تكرار الجلب**: كل 15 دقيقة / 30 / ساعة
- **عمر الخبر الأقصى**: 24 ساعة / 48 / أسبوع
- **المصادر**: فعّل أو عطّل أي مصدر، أو أضف مصدراً جديداً

***

## 🐛 المشاكل والحلول
| المشكلة | الحالة | الحل |
|---------|--------|------|
| خطأ `API_KEY_INVALID` من Gemini | **محلولة** | تم استبدال مفتاح API في ملف `.env` وعاد الجلب للعمل بشكل صحيح. |
| `setup.bat` لا يعمل | - | تأكد أن Python مثبت: <https://python.org/downloads> |
| المتصفح لا يفتح | - | افتح يدوياً: <http://127.0.0.1:8000> |
| لا تظهر أخبار | - | اضغط "جلب الأخبار الآن" يدوياً |
| المنفذ 8000 مشغول | - | غيّر `PORT=8001` في ملف `.env` |

## 📁 هيكل الملفات

```
murrasil/
├── 🟢 start.bat        ← شغّل التطبيق
├── 🔧 setup.bat        ← الإعداد (مرة واحدة)
├── 🔴 stop.bat         ← أوقف التطبيق
├── 📄 .env             ← مفاتيح API (لا ترفعه على GitHub)
├── 📄 .env.example     ← نموذج .env (آمن للمشاركة)
├── 🗃️ news.db          ← قاعدة البيانات المحلية
└── 📖 UserReport.md    ← هذا الملف
```

***

## ⚠️ تنبيهات مهمة

- **لا ترفع ملف `.env` على GitHub** — يحتوي على مفتاحك السري
- ملف `news.db` يحفظ كل بياناتك محلياً — احتفظ بنسخة احتياطية منه
- الخطة المجانية لـ Gemini تتيح **1500 طلب يومياً** — أكثر من كافٍ
